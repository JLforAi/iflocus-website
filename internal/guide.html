SurveyAI 安裝與操作指南

問卷模擬與自動填寫工具 · 適用 Google Forms / SurveyCake

0這個工具是什麼

SurveyAI 是 iFLocus 的內部研究工具，支援兩種工作模式：

模式做什麼適合情境
💬 AI 模擬回答 產生 Prompt，貼到 Claude / ChatGPT，AI 輸出模擬問卷回答 快速產出分析資料、無需真實 IP
🖥️ 自動填寫表單 Python 腳本自動點擊填寫，Google Forms 後台出現真實筆數 需要真實送出紀錄、多人分工執行
💡 工具網址：iflocus.com/internal/survey.html　·　密碼請洽主管

📝 各步驟說明
  • Step 1 — 匯入需求單
    下載 Excel 範本，填入品牌名稱、產品名稱、問卷連結等基本資訊後上傳。或點「使用範例資料」快速體驗。
  • Step 2 — 受訪者條件設定
    設定性別、年齡、居住地、職業、收入等篩選條件。未指定的欄位由 AI 自由設定。「姓名顯示方式」可選完整姓名、姓氏縮寫（王OO）或不顯示。
  • Step 3 — 選擇執行方式與份數
    選「AI 模擬回答」，再選模擬模式（純AI 或 樣本擴增）與份數。排程設定在此模式下不需理會。
  • Step 4 — 複製 Prompt 並執行
    點金色按鈕「📋 複製完整 Prompt」，開啟 Claude.ai 或 ChatGPT，直接貼上送出。AI 會產出模擬問卷回答並附上可下載的 CSV 檔案。
  • 取得分析檔案
    若 AI 平台有附檔功能，直接下載 CSV。若沒有，複製 AI 回覆貼入工具下方的「AI 回覆轉下載檔」區塊，點「產生下載檔與分析報表」即可下載。
2自動填寫表單（需安裝 Python）
⚠️ 每台電腦只需安裝一次，約 5–10 分鐘。安裝完成後，之後每次執行只需雙擊 .bat 檔即可。
🔧 第一步：安裝 Python
  • 前往 python.org 下載
    搜尋「python download」，進入官網下載 Python 3.11 或以上版本（Windows Installer）。
  • 安裝時勾選「Add Python to PATH」
    這個選項在安裝畫面最下方，一定要勾選，否則後續指令會找不到 Python。
  • 確認安裝成功
    開啟命令提示字元（cmd），輸入以下指令，若出現版本號即成功：
    python --version
🔧 第二步：安裝 Playwright（瀏覽器自動化套件）
  • 開啟命令提示字元（cmd）
    按 Win + R，輸入 cmd，按 Enter。
  • 貼上以下指令並按 Enter
    pip install playwright && python -m playwright install chromium
    等待安裝完成（約 3–5 分鐘），看到沒有 Error 訊息即成功。
📁 第三步：準備執行檔案
  • 建立一個資料夾
    在桌面或任意位置建立資料夾，例如 SurveyAI。
  • 下載 survey_auto_runner.py
    從主管取得 survey_auto_runner.py，放入剛才建立的資料夾。
  • 從工具下載設定檔 JSON
    在 SurveyAI 工具的 Step 4 自動填寫區，點「⬇ 下載設定檔 JSON」，將檔案放到同一個資料夾。
  • 從工具下載 .bat 執行檔
    工具會依人數產生對應的 .bat 檔（例如 run_survey_同事A.bat），點「⬇ 下載 .bat」，放到同一個資料夾。
📁 最終資料夾內容應為：
SurveyAI/
├── survey_auto_runner.py
├── 案件名稱-settings.json
└── run_survey_同事A.bat
▶ 第四步：執行
  • 雙擊 .bat 檔
    黑色視窗會自動開啟，顯示執行進度。瀏覽器會在背景自動填寫問卷（headless 模式，不會看到瀏覽器視窗）。
  • 等待完成
    每份問卷間隔 30–90 秒（模擬真實填寫），完成時視窗顯示「執行完成」並自動停住。
  • 查看報告
    執行完成後，資料夾內會自動產生 survey_auto_results/ 子資料夾，裡面有每次執行的 CSV 報告與截圖。
⚠️ 執行期間電腦不可睡眠、不可斷網。執行完成後電腦可正常使用。
3多人分工執行（分散 IP）
🗂 為什麼需要多人分工

Google Forms 會偵測同一 IP 大量提交，建議每個 IP 不超過 30 份。三人各自在家使用不同網路執行，等同使用三個不同住宅 IP，可大幅降低被系統偵測的風險。

👥 分工流程
  • 主管在工具設定人數
    在 Step 4 自動填寫區填入問卷網址，「執行人數」填 3（或實際人數）。工具自動計算每人份數。
  • 下載各自的 .bat 檔
    工具會產生「同事 A」「同事 B」「同事 C」各自的執行指令與 .bat 檔，每人的任務範圍自動切分、不重疊。
  • 傳給對應同事
    將對應的 .bat 檔 + settings.json + survey_auto_runner.py 傳給各人（Line / Email 均可）。
  • 各自在家執行
    各人把檔案放在同一個資料夾，雙擊自己的 .bat 執行。建議約好同一天晚上各自在家執行，約 30–50 分鐘完成。
  • 回傳 CSV 報告
    執行完成後，將 survey_auto_results/ 內的 CSV 回傳給主管彙整。
✅ 重複提交保護：就算不小心重複執行，系統會自動跳過已送出的份數（顯示 skipped_duplicate），不會重複計算。
4常見問題
問題解法
執行後出現「python 不是內部或外部命令」 Python 安裝時未勾選「Add to PATH」，重新安裝並勾選
出現「No module named playwright」 重新執行安裝指令：pip install playwright && python -m playwright install chromium
執行後視窗閃一下就關掉 雙擊 .bat 前先確認三個檔案在同一個資料夾；檢查 settings.json 檔名是否正確
Google Forms 出現驗證碼（CAPTCHA） 停止執行，等隔天再繼續；或減少每人份數、拉長間隔時間
執行到一半電腦睡眠了 重新雙擊 .bat，系統會自動跳過已完成的部分，從中斷點繼續
CSV 報告在哪裡 執行資料夾內的 survey_auto_results/ 子資料夾
問題請洽 iFLocus 內部研究團隊
